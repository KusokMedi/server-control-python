# WebControl

WebControl is a web-based interface for controlling a computer remotely. It provides full access to file management, process control, system monitoring, window management, keyboard/mouse input, screenshots, clipboard, and logs.

## Features

- **File Manager**: Browse, create, delete, rename files and folders. Edit text files, download/upload, run executables.
- **Processes**: View and manage all system processes.
- **Monitoring**: Live CPU, RAM, Disk, Network stats.
- **Windows**: Control desktop windows (hide/show, minimize/maximize, move, focus).
- **Input**: Simulate keyboard and mouse actions.
- **Screenshots**: Capture screen images.
- **Clipboard**: Read/write/clear clipboard text.
- **Logs**: View system action logs.

## Installation

1. Install dependencies: `pip install -r requirements.txt`
2. Set password in `admin.password`
3. Run: `python run.py` or use `start.bat`

## Usage

Access http://localhost:5000, login with password from admin.password.

Use the sidebar to navigate modules.
